## 1. 系统概述

**企业管理系统**是一套基于 Vue 3 + Vite + Ant Design Vue 构建的一体化企业经营管理平台，涵盖客户、项目、采购、库存、财务与办公流程的统一管理。

### 系统特性

- 📊 **数据可视化**：经营驾驶舱、风险中心、业务待办一站式管理
- 🔄 **业务闭环**：从商机到合同、采购到付款、项目到成本的完整业务链路
- 🔒 **权限控制**：基于角色的细粒度权限管理
- 📱 **响应式设计**：支持桌面端和移动端访问

### 技术栈

- Vue 3.5 + TypeScript
- Vite 8.1
- Ant Design Vue 4.2
- Pinia 状态管理
- Vue Router 4.5

## 2. 登录与权限

### 2.1 登录入口

访问系统首页会自动跳转到登录页面：`http://localhost:5173/login`

### 2.2 默认账号

| 账号 | 密码 | 角色 |
|------|------|------|
| admin | Admin@123 | 系统管理员 |

### 2.3 登录流程

1. 在登录页面输入账号和密码
2. 点击「登录系统」按钮
3. 系统验证通过后自动跳转至客户池页面

### 2.4 权限说明

系统采用基于角色的访问控制（RBAC），不同角色拥有不同的功能权限：

- **系统管理员**：拥有所有模块的访问和操作权限
- **销售角色**：可访问CRM模块相关功能
- **采购角色**：可访问采购模块相关功能
- **项目角色**：可访问项目管理模块相关功能
- **财务角色**：可访问财务模块相关功能

### 4.2 CRM客户关系管理

**功能定位**：管理客户信息、商机跟踪、合同管理、应收款等全流程。

#### 4.2.1 模块结构

```
CRM
├── CRM仪表盘        ← CRM数据总览
├── 客户池           ← 客户列表管理
├── 线索商机         ← 商机跟踪
├── 报价方案         ← 报价管理
├── 客户合同         ← 合同管理
├── 跟进回访         ← 客户跟进记录
├── 续约管理         ← 合同续约提醒
└── 合同应收         ← 应收款管理
```

#### 4.2.2 客户池

**客户池页面**是CRM模块的核心，提供客户信息的统一管理。

##### 操作指南

**新增客户**：
1. 点击「新增客户」按钮
2. 填写基本资料（客户名称、行业、等级、负责人等）
3. 填写开票资料（发票抬头、税号、银行信息等）
4. 添加联系人（姓名、职务、电话、邮箱）
5. 添加项目地址
6. 点击「确定」保存

**查看客户详情**：
1. 点击客户列表中的客户名称或「查看」按钮
2. 在右侧抽屉中查看客户详细信息
3. 可切换查看：客户档案、跟踪项目、已签合同、合同应收、跟进记录

**编辑客户**：
1. 在客户详情抽屉中点击「编辑档案」
2. 修改相关信息
3. 点击「保存」

**删除客户**：
1. 在客户列表中点击「删除」按钮
2. 确认删除操作（将删除所有关联数据）

**搜索与筛选**：
- 支持按关键词搜索（客户名称、编码、行业、负责人）
- 支持按客户等级筛选（战略客户、重点客户、普通客户）
- 支持按风险状态筛选（正常、逾期、续约风险）

**导出功能**：
1. 点击「导出」按钮
2. 系统自动下载CSV格式的客户数据

### 4.3 供应链采购

**功能定位**：管理采购申请、订单、入库、供应商等全流程。

#### 4.3.1 模块结构

```
供应链采购
├── 采购申请         ← 发起采购申请
├── 采购订单         ← 采购订单管理
├── 到货入库         ← 入库管理
├── 成本归集         ← 成本分配
├── 采购应付         ← 应付款管理
├── 供应商           ← 供应商管理
├── 采购分析         ← 采购数据分析
└── P2P流程         ← 采购全流程追踪
```

#### 4.3.2 采购申请

**采购申请页面**用于发起新的采购请求。

##### 操作指南

**新增采购申请**：
1. 点击「新增申请」按钮
2. 填写物料信息（物料名称、规格型号）
3. 填写数量和单位
4. 设置需求日期
5. 填写预计单价
6. 选择成本归属（项目采购或部门采购）
7. 选择关联项目或部门
8. 填写申请原因
9. 点击「确定」提交

**从补货建议创建**：
1. 在库存补货计划区域查看建议
2. 点击需要补货的物料卡片
3. 系统自动填充采购申请表单
4. 修改确认后提交

**提交审批**：
1. 在草稿状态下点击「提交审批」
2. 等待审批人处理

### 4.4 项目管理

**功能定位**：管理项目立项、预算、成本、阶段推进等全流程。

#### 4.4.1 模块结构

```
项目管理
├── 项目列表         ← 项目概览

└── 阶段履历         ← 阶段变更记录
```

#### 4.4.2 项目列表

**项目列表页面**提供项目的整体视图。

##### 操作指南

**新增项目**：
1. 点击「新增项目」按钮
2. 填写项目基本信息（名称、类型、客户、负责人等）
3. 设置计划周期和质保截止
4. 填写合同金额和预算金额
5. 提交立项审批

**项目详情**：
1. 点击项目名称或「详情」按钮
2. 查看项目基本信息、预算执行、成本明细、阶段履历

**立项审批**：
1. 找到审批状态为「待审批」的项目
2. 点击「审批」按钮
3. 选择通过或驳回，填写审批意见

**推进阶段**：
1. 在项目详情中点击「推进阶段」
2. 选择下一阶段
3. 填写节点说明

**登记成本**：
1. 在项目详情中点击「登记成本」
2. 选择成本分类（人工、材料物料、外包、差旅等）
3. 选择来源（手工登记、仓库领料、费用报销等）
4. 填写成本金额和说明

## 5. 常见操作指南

### 5.1 如何提交请假申请

1. 点击左侧菜单「员工自助」→「我的工作台」
2. 点击「提交请假」按钮（或导航至 `/self/leaves/new`）
3. 选择请假类型（年假、病假、事假等）
4. 填写请假起止日期
5. 填写请假原因
6. 点击「提交」等待审批

### 5.2 如何创建采购申请

1. 点击左侧菜单「供应链采购」→「采购申请」
2. 点击「新增申请」按钮
3. 填写物料信息、数量、需求日期
4. 选择成本归属（项目采购或部门采购）
5. 填写申请原因
6. 点击「确定」创建申请（草稿状态）
7. 在列表中点击「提交审批」

### 5.3 如何查看客户合同详情

1. 点击左侧菜单「CRM」→「客户池」
2. 在客户列表中找到目标客户
3. 点击客户名称打开详情抽屉
4. 切换到「已签合同」标签页
5. 查看合同列表和详细信息

### 5.4 如何处理审批任务

1. 点击左侧菜单「业务待办中心」，切换到「审批处理」
2. 在待审批列表中找到需要处理的审批
3. 点击查看审批详情
4. 点击「通过」或「驳回」并填写意见
5. 提交审批结果

```
   │                                          │
   └────────────── 成本归集 ←──────────────────┘
```

```
商机 → 报价 → 合同签订 → 合同应收 → 开票 → 回款 → 核销
   ↑                │
   │                └── 续约管理
   └── 跟进回访
```
