## 4. 剩余失败项

| 步骤 | HTTP | 判断 |
| --- | --- | --- |
| 总经办审批合同 | 400 | 业务规则：后端要求先上传合同审批件 `APPROVAL_DOC`。本轮 API 未上传附件，不算接口缺失。 |
